The Food Wastage Reduction Application is designed to minimize food waste by allowing individuals and restaurants to donate excess food to those in need. Medicine Shop Automation

This web application streamlines pharmacy operations by automating inventory management, sales, and billing using Java Servlets, JDBC, and SQL. It ensures efficient stock control, accurate sales tracking, and enhanced customer service for shop owners and customers alike.

Technologies: Java Servlets, JDBC, HTML&CSS, JavaScript, SQL

Fake Job Prediction Application

A machine learning model that predicts whether a job posting is fake or real. It leverages text features (e.g., title, description) and numerical indicators (e.g., telecommuting flag, company logo) in a hybrid ensemble of Logistic Regression, Random Forest, and Gradient Boosting with a Stacking Classifier to boost accuracy.

Technologies: Python, Machine Learning, Logistic Regression, Random Forest, Gradient Boosting

Key Logger (Python)
